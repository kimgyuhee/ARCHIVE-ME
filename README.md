## 🚀 사용 방법
1. GitHub에서 저장소를 **클론(Clone)** 하거나 **다운로드**  
2. 브라우저에서 `index.html` 파일 열기  
3. [GitHub Pages](https://pages.github.com/)로 호스팅하면 바로 웹사이트 확인 가능
